# @webstackbuilders/backstage-ai-core-backend-module-cloud-providers-kubernetes

The Kubernetes extension for cloud providers backend module for the ai-core plugin.

- Set up a **Vitest integration test backend runner** verifying the connection chain from the `cloud-providers` core to this sub-plugin module.
- Configure **SigV4 authorization headers** if the driver needs to communicate through an AWS EKS credential proxy.
